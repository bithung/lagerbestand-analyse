# Lagerbestand-Analyse mit Python

Kleines Python-Projekt zur Auswertung von Lagerbeständen aus einer CSV-Datei.

## Funktionen
- Einlesen einer CSV-Datei
- Prüfung von Mindestbeständen
- Ausgabe von Artikeln mit Nachbestellbedarf

## Verwendete Technologien
- Python
- CSV-Datei

## Warum dieses Projekt?
Ich komme aus der Lagerlogistik und lerne aktuell Python.
Mit diesem kleinen Projekt möchte ich Logistikpraxis und IT-Interesse verbinden.

## Beispiel Ausgabe

=== Lagerbestand Analyse ===

Karton klein (Bestand: 25, Mindestbestand: 30)

Scannerakku (Bestand: 5, Mindestbestand: 10)

=== Zusammenfassung ===
Geprüfte Artikel: 5
Artikel unter Mindestbestand: 3
